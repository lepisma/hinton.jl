# hinton

Create hinton diagrams in Julia.

Hinton diagrams are used to visualize weight matrices in neural networks. Here is a pure Julia/Tk implementation.

### Usage

```julia
Pkg.clone("git://github.com/lepisma/hinton.jl.git")
```

```julia
using hinton

hintondiag(matrix)
```
